# 🌐 Language Translation Tool

## 📌 Project Overview

The Language Translation Tool is a Python-based application that translates text from one language to another using machine translation techniques. The project provides a simple and user-friendly interface where users can enter text, select source and target languages, and instantly receive translated output.

This tool helps overcome language barriers and enables seamless communication across different languages.

---

## 🚀 Features

- Translate text between multiple languages
- User-friendly interface
- Fast and accurate translations
- Supports various international languages
- Easy to use and lightweight
- Real-time translation output

---

## 🛠 Technologies Used

- Python
- Google Translate API / deep-translator
- Tkinter (if GUI used)
- HTML, CSS, JavaScript (if web-based)

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/swapna-nair/CodeAlpha_Language_Translation_Tool.git
cd CodeAlpha_Language_Translation_Tool
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

### For Python Application

```bash
python translator.py
```

### For Flask Web Application

```bash
python app.py
```

Then open:

```
http://127.0.0.1:5000
```

in your browser.

## 🧠 How It Works

1. User enters text to be translated.
2. Selects source and target languages.
3. The application sends the text for translation.
4. The translated text is returned and displayed instantly.
